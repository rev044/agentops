# v2.35.0

## Highlights

The CLI monolith decomposition is complete. Waves 5-13 extracted business logic from `cmd/ao` into 15 `internal/` domain packages, deleted deprecated files, and enabled test shuffle in CI to catch ordering bugs. The knowledge compiler skill (`/compile`) ships as a new capability.

## What's New

- **Knowledge compiler** — `/compile` skill with incremental Karpathy-style compilation and pluggable LLM backends (ollama or claude)
- **Dependency injection** — App struct replaces mutable package-level globals across gc bridge, events, executor, context, and stream modules
- **Test shuffle in CI** — `-shuffle=on` catches ordering-dependent tests; 6 fixed in this release

## All Changes

See [CHANGELOG.md](CHANGELOG.md) for the complete list of additions, changes, fixes, and removals.

## Removed

- `fire.go` and `rpi_workers.go` (superseded by gc bridge)
- `ao rpi workers` subcommand
- Coverage percentage CI gates (`coverage-ratchet`, `check-cmdao-coverage-floor`)
