# AgentOps v2.18.2

Hotfix release closing gaps in `ao seed` and the MemRL feedback loop.

## Highlights

- **`ao seed` now bootstraps correctly** -- creates `.gitignore`, storage directories, and uses the current MEMORY.md + session hooks paradigm instead of stale `ao inject`/`ao forge` references
- **MemRL feedback loop closed** -- `ao feedback-loop` command wired end-to-end, `ao maturity --recalibrate` dry-run guard prevents accidental bulk promotion
- **Housekeeping** -- `.agents/` session artifacts removed from git tracking, PRODUCT.md refreshed (Olympus section removed), GOALS.md coverage target updated to measured 78.8%

## What's New

### Fixed

- `ao seed` now calls `setupGitProtection`, `ensureNestedAgentsGitignore`, and `initStorage` from `ao init` -- new repos get proper `.gitignore` and storage directories on first seed
- `ao seed` help text and bootstrap learning updated from stale `ao inject`/`ao forge` references to current MEMORY.md + session hooks paradigm
- `ao feedback-loop` command wired and documented in CLI reference
- `ao maturity --recalibrate` now requires `--dry-run=false` to apply changes (safe by default)
- Quickstart skill updated to reference `ao seed` and current flywheel docs

### Changed

- `.agents/` session artifacts removed from git tracking (were accidentally committed before gitignore rule)
- PRODUCT.md updated -- Olympus section removed, value props and skill tier counts corrected
- GOALS.md coverage directive updated to measured 78.8% (target 85%)
