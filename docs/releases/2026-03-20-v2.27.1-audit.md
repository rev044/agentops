## Root Cause

Golden signals feature shipped in v2.27.0 behind `--golden` flag. The default `ao flywheel status` output reported "COMPOUNDING" based on the simple sigma-rho > delta check, while the comprehensive golden signals analysis (velocity trend, citation pipeline, research closure, reuse concentration) concluded "accumulating." Users never saw the honest assessment unless they knew about the hidden flag.

## Changes

- `cli/cmd/ao/metrics_flywheel.go` — Removed `flywheelGolden` gate, always compute golden signals, hidden `--golden` flag
- `cli/cmd/ao/metrics_flywheel_test.go` — Renamed test to verify golden signals appear without flag
- `skills/flywheel/SKILL.md` — Updated docs, trimmed to 300-line limit
- `README.md`, `docs/knowledge-flywheel.md` — Removed `--golden` flag references
- `cli/docs/COMMANDS.md` — Regenerated CLI reference
