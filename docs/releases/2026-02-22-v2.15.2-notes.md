# v2.15.2 Release Notes

## Highlights

- **Hooks stripped to bare minimum** -- 22 hooks across 12 events reduced to 3 hooks across 3 events
- **Only the flywheel survives** -- extract, inject, forge, maturity, close-loop. Everything else cut.
- **session-start.sh: 600 -> 100 lines** -- removed handoff recovery, ratchet display, chain rotation, environment manifest, prune check

## Why

Hooks were breaking sessions and adding too much friction. The knowledge flywheel is the only thing that needs to run automatically. Safety gates, nudges, and telemetry hooks can be re-enabled manually if needed -- the scripts are still in `hooks/`.

## What Changed

### Removed from hook registration (scripts preserved)

- **PreToolUse** (5 hooks): standards-injector, skill-lint-gate, git-worker-guard, dangerous-git-guard, pre-mortem-gate
- **PostToolUse** (2 hooks): ratchet-advance, citation-tracker
- **UserPromptSubmit** (2 hooks): context-guard, prompt-nudge
- **TaskCompleted** (1 hook): task-validation-gate
- **Stop** (2 hooks): stop-team-guard, stop-auto-handoff
- **PreCompact, SubagentStop, WorktreeCreate, WorktreeRemove, ConfigChange**: all removed

### Simplified

- **session-start.sh**: Kept mkdir, gitignore, `ao extract`, `ao inject`, skill context injection. Cut everything else.
- **session-end-maintenance.sh**: Kept `ao forge transcript` and `ao maturity`. Cut session-outcome, feedback-loop, task-sync, batch-feedback, vibe-check.

### What remains (3 hooks, 3 events)

| Event | Hook | Purpose |
|-------|------|---------|
| SessionStart | session-start.sh | mkdir .agents, extract pending, inject knowledge, skill context |
| SessionEnd | session-end-maintenance.sh | forge transcript, maturity scan/expire/evict |
| Stop | ao-flywheel-close.sh | flywheel close-loop |
