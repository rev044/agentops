# retro

Capture implementation learnings in a Codex-first post-work format.

## Codex Execution Profile

1. Treat `skills/retro/SKILL.md` as canonical retrospective workflow.
2. Prefer repository-local artifacts (`.agents/`, `docs/`, issue history) over runtime-specific paths.
3. Produce findings that map directly to process or code changes.

## Guardrails

1. Avoid assumptions about `.claude/*` command layouts.
2. Classify each finding by severity and recurrence risk.
3. Add one concrete prevention action per high-severity finding.
