# dream

Private overnight operator mode. Routes interactive Dream requests to the
shared `ao overnight` engine for setup, bedtime runs, and morning reports.
Triggers: "dream", "overnight", "bedtime run", "morning report",
"dream setup", "dream report", "dream council", "dreamscape".

## Instructions

Load and follow the skill instructions from the sibling `SKILL.md` file for
this skill. Then read repo docs or CLI help when you need the concrete Dream
contracts or command flags.
