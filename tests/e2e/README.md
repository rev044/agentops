# End-to-End Tests

Flywheel proof-run harness that demonstrates three-session compounding of the AgentOps knowledge flywheel: discovery (research and learn), compounding (inject learnings), and maturation (re-inject to show growth). Fully automated with no interactive prompts, CI-runnable, and uses a temporary work directory with fixture data.

## Running

```bash
bash tests/e2e/proof-run.sh
```
